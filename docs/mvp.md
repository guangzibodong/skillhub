# MVP Plan

## Milestone 1: Local Product Skeleton

- Monorepo.
- Manifest schema.
- Example skill.
- Web shell.
- Gateway routes.
- SDK and CLI foundations.

## Milestone 2: Hosted Registry

- Supabase database.
- GitHub repository.
- Vercel or Cloudflare Pages deployment.
- Cloudflare Worker deployment.
- API key issuance.

## Milestone 3: Agent Integrations

- MCP discovery endpoint.
- OpenAPI export.
- TypeScript SDK publish.
- CLI publish flow.

## Milestone 4: Trust Layer

- Verification queue.
- Permission risk labels.
- Audit logs.
- Basic eval runner.
